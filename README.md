# 📦 Warehouse Analytics Dashboard
### Power BI | Supply Chain | Inventory Intelligence  

---

## 🚀 Project Overview

This project presents a professionally designed **Warehouse Analytics Dashboard** built using Power BI to monitor and analyze warehouse operations in real time.

The dashboard provides actionable insights into **inventory, inbound/outbound flow, fulfillment performance, and operational efficiency**, supporting data-driven decision-making in supply chain management.

---

## 🎯 Key Objectives

- Track inventory levels and warehouse performance  
- Monitor inbound and outbound operations  
- Evaluate order fulfillment and shipping efficiency  
- Identify stock issues and operational bottlenecks

## 📊 Key Metrics (KPIs)

- Total Inventory Value  
- Total Inbound & Outbound Orders  
- Order Fulfillment Rate  
- Inventory Turnover  
- Stock Accuracy  
- On-Time Shipping Performance

## 📌 Features & Analysis

### ✅ Inventory Management
- Inventory value distribution by category  
- Stock status (In Stock, Low Stock, Out of Stock)  
- Inventory heatmap across warehouses  

---

### ✅ Operations & Flow
- Inbound vs Outbound trends over time  
- Top fast-moving products  
- Warehouse performance comparison  

---

### ✅ Order & Delivery Insights
- Orders by status (Completed, Pending, Cancelled)  
- On-time shipping performance tracking  
- Fulfillment efficiency analysis  

---

## 📈 Key Insights

- Inventory value increased year-over-year, indicating growth in stock levels  
- Outbound operations show higher activity than inbound, suggesting strong product movement  
- Identified low-stock and out-of-stock items that require immediate attention  
- High stock accuracy reflects reliable inventory tracking processes
